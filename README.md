# meimakes.com

Personal website for Mei Park - Building and shipping iOS apps in naptime sprints.

## About

This is the source code for [meimakes.com](https://meimakes.com), a personal website showcasing my journey from 10-year software engineer to stay-at-home mom building iOS apps during naptime.

## Tech Stack

- HTML/CSS/JavaScript
- Tailwind CSS (via CDN)
- Lexend font family
- Dark mode support with CSS custom properties
- Responsive design

## Features

- 🌓 Automatic dark/light mode based on system preferences
- 📱 Mobile-responsive design
- 📊 GitHub contribution chart integration
- 🔗 Social media metadata for rich link previews

## Local Development

Simply open `index.html` in your browser or use a local server:

```bash
python3 -m http.server 8000
# or
npx serve .
```
