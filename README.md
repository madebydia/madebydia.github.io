# meimakes.com

Personal homepage for [meimakes.com](https://meimakes.com) — a landing page for two newsletters and everything I build.

Built by [@meimakes](https://x.com/meimakes)

## Features

- **Two-card layout** — The Undercurrent (builder intelligence) + Raising Pixels (teaching kids to think like builders)
- **IBM Plex Mono** throughout for that terminal/builder aesthetic
- **Light/dark mode** via `prefers-color-scheme` (no toggle, just respects your system)
- **Hard drop shadows** at different angles on each card for personality
- **Brand colors** — #FF4858 coral (subscribe buttons), #14B9A7 teal (Raising Pixels + ebook button)
- **Profile image** links to [chat.meimakes.com](https://chat.meimakes.com)
- **Structured data** — JSON-LD Person schema for LLM and search engine discovery
- **Plausible analytics** — privacy-friendly, no cookies
- **Responsive** — stacks to single column on mobile
- **Minimal** — single HTML file, no build step, no dependencies beyond Google Fonts
